# CivicEvents+ Demo Guide

This guide helps you demonstrate all features of the CivicEvents+ platform for the required 5-7 minute video demo.

## üé¨ Demo Script (5-7 minutes)

### 1. Backend Setup (30 seconds)
- Show terminal with backend running: `npm run dev`
- Briefly show database connection success message
- Navigate to `http://localhost:4000` to show API is running

### 2. Frontend Setup (30 seconds)
- Show frontend being served: `npx serve .`
- Open browser to `http://localhost:3000`
- Show the landing page with login/signup forms

### 3. User Registration & Authentication (1 minute)
- **Signup Flow**:
  - Fill out signup form with strong password
  - Show password strength meter in action
  - Demonstrate validation (try weak password first)
  - Successfully create account
- **Login Flow**:
  - Login with created credentials
  - Show "Keep me signed in" option
  - Demonstrate successful authentication

### 4. Core User Features (2 minutes)
- **Events**:
  - Browse events list with filters
  - Click on an event to view details
  - Register for an event
  - Show registration confirmation
  - Leave feedback with rating and comment

- **Media Consumption**:
  - Navigate to Announcements
  - Play an audio announcement (show accessible controls)
  - Navigate to Promos
  - Play a video promo with captions

- **Service Requests**:
  - Navigate to Services
  - Create a new service request
  - Show form submission and confirmation

- **Profile & Notifications**:
  - Check notifications bell (show count)
  - Open notifications drawer
  - Visit profile page and update information

### 5. Admin Features (2 minutes)
- **Login as Admin**:
  - Logout and login with admin credentials
  - Show admin navigation appears

- **Dashboard**:
  - Show admin dashboard with metrics
  - Explain the various statistics displayed

- **Content Management**:
  - Create a new event with image upload
  - Show image preview functionality
  - Publish the event

- **Media Management**:
  - Upload an audio announcement
  - Upload a video promo with captions
  - Show admin-only controls

- **User Management**:
  - Navigate to user management
  - Show user list with filters
  - Enable/disable a user account
  - View user details in modal

- **Notifications**:
  - Create and send a broadcast notification
  - Show targeting options (all users, admins only, etc.)

### 6. Accessibility & Responsiveness (1 minute)
- **Accessibility**:
  - Navigate using only keyboard (Tab key)
  - Show focus indicators
  - Demonstrate screen reader compatibility (if available)

- **Responsive Design**:
  - Resize browser window to show mobile layout
  - Show navigation adapts to smaller screens
  - Demonstrate touch-friendly interface

- **Additional Features**:
  - Use global search functionality
  - Show loading states and error handling
  - Demonstrate offline detection (disconnect internet briefly)
